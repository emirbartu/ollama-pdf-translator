Full self-hosting guide

  1. 1

    Install Ollama

    Download from ollama.com — macOS and Windows have desktop apps, Linux uses a one-liner:

    curl -fsSL https://ollama.com/install.sh | sh
  2. 2

    Pull a model

    Any chat model works. Start with a small, fast one:

    ollama pull llama3.2

    Other options: ollama pull qwen2.5 or ollama pull mistral.

  3. 3

    Get the code

    Clone the repository and install dependencies with uv (or use pip — see note below):

    git clone https://github.com/emirbartu/ollama-pdf-translator.git && cd ollama-pdf-translator && uv sync

    Prefer pip? Run pip install openai python-dotenv PyMuPDF tqdm fastapi "uvicorn[standard]" python-multipart jinja2 slowapi instead.

  4. 4

    Translate from the terminal

    Point it at a PDF and pick your languages:

    python main.py input.pdf -s English -t Turkish -m llama3.2
  5. 5

    Run this web UI

    Launch the server, then open it in your browser:

    uvicorn app:app

    Open http://127.0.0.1:8000 — the model picker above works right here once Ollama is running.

  6. 6

    Optional: Ollama on another machine

    Run Ollama on a beefy server and point this app at it via .env:

    OPENAI_BASE_URL=http://your-server:11434/v1
  7. 7

    Optional: any OpenAI-compatible API

    Use OpenAI, OpenRouter, LM Studio, or any compatible endpoint — set both variables in .env:

    OPENAI_BASE_URL=https://api.openai.com/v1
OPENAI_API_KEY=sk-your-key-here
